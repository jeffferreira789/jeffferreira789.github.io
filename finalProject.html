This is a virtual Connect Four game made for two players that can be won by getting four chips of the same
color in a row either horizantally or vertically. Players will start by pressing the start button on the front page.
they will then take turns clicking on the empty white chip slots. Chips can be placed anywhere on the 6 by 7 gameboard.
When the first player clicks a red chip will be placed. The next player place a yellow chip in there slot of choice.
Then it will switch back to the red players turn and so on. Below is a video showing how the game works and another video
of how the code works.
